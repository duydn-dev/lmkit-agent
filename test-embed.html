Website Bất Kỳ (Test Host)

Đây là một trang web mô phỏng để kiểm tra tính năng nhúng Chat Widget giống Zalo.

Bạn sẽ thấy một nút chat nổi (floating button) ở góc dưới cùng bên phải. Hãy click vào nó để mở khung chat lên.

Đảm bảo rằng Backend API và Frontend (Vue) đang chạy tại http://localhost:5173.
